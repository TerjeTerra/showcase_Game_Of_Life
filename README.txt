-- GAME OF LIFE --
Et prosjekt skrevet som obligatorisk innlevering våren 2023,
Universitet i Oslo,
av Terje Haldorsen

Startes fra filen Hovedprogram (når kompilert)
Kan oppgi egendefinert størrelse på vindu, eller velge standard.

Spillet er en visualisering av Game of Life, der celler (@) kan 
formere seg eller dø etter bestemte regler basert på sine nabo-
celler. Man kan klikke på celler for å endre status (død eller
levende), og man kan pause og starte spillet igjen.
